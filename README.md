## Development

- Install dependencies:

```bash
pnpm install
```

- Develop the library with demo:

```bash
pnpm run dev
```

- Run the unit tests:

```bash
pnpm run test
```

- Build the library:

```bash
pnpm run build
```

- Build the demo:

```bash
pnpm run build:demo
```
